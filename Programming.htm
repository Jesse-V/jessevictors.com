Programming

I have a passion for technology and computer science. I consider computers and the Internet to be mankind's greatest invention. It continues to impress me how powerful and small computer components actually are, (CPUs are built on the nanosecond scale!) and I've always been intrigued by the new ways in which we are putting them to use. Nearly everyone around me has their own computer. Behind every one of them is more than just code: the operating system and every program solve so many problems for us, and most of the time we take it for granted. How is that you can run multiple applications at the same time? How can the hard drive—an relatively slow medium—efficiently respond to multiple requests for data, or concurrently handle several data transfers that are occuring all at once? There are just some of the difficult problems that computer scientists solve. After a lot of thinking, discussion, planning, and testing, we come up with really clever solutions like the LOOK algorithm and Completely Fair CPU Scheduler. I really enjoy deep problem solving like that, and looking at problems from a lot of different angles.

I know a number of different programming languages, some at the application level such as Java, C++, and MIPS assembly, and others designed more for the web, like HTML5, CSS3, LESS, Javascript, and PHP. I've worked with Javascript technologies like Node.js, JSON, JQuery, Handlebars, and Backbone. Each language has a purpose, and when used together the product is really amazing. A number of times I'll run across a problem and realize that I could write a program to solve it for me faster than I could solve the problem directly myself. Sudoku is an example. Half an hour later, and my recursive backtracking algorithm can solve the puzzle in general. :) Obviously, some puzzles are more fun to solve by hand, such as the Rubik's Cube.

List of programs coming soon! In the meantime, check out my programming activity on Github. Here are several software projects I'm particuly proud of: SpeedyShop, Victory Paint, and a renderer for the Mandelbrot fractal.
